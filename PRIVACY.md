## 3. Data Storage and Security

### 3.1 Cloud Storage (Firebase Firestore)

Your data is stored securely in Google Firebase Cloud Firestore:

**Stored Data:**
- User profile (email, display name)
- Quit data (quit date, cigarettes per day, costs)
- Achievements unlocked
- Goals and history
- FCM notification tokens
- Last seen timestamp

**Security Measures:**
- All data transmitted via HTTPS (encrypted)
- Firebase Authentication for secure access
- Firestore security rules restrict access to your own data only
- No user can access another user's data

### 3.2 Local Storage

Some data is temporarily cached on your device for:
- Offline functionality
- Faster app performance
- Better user experience

**This includes:**
- User preferences
- Recent achievements
- Cached statistics
